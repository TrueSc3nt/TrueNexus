# TrueNexus Directory

In-app encyclopedia of **every** mode, flag, pattern, BSGS strategy, mnemonic submode, GUI tab, and research idea.

Open the **Directory** sidebar tab → Search or Show ALL.

Generated from:

- `truenexus/directory.py`
- `truenexus/ideas_catalog.py`
- TrueCollider CLI (`docs/COMMANDS.md` / `-h`)

## How to use

1. New user: Directory → search your goal (e.g. `partial seed`, `pubkey`, `vanity`)
2. Jump to the matching **Lab** tab (Mnemonic, Kangaroo, Shadow160, …)
3. Select the idea → **Apply → TrueCollider** → Dry-Run (`-y`) → Launch
4. Pros: TrueCollider tab has every flag on one page
